# Amazon-sales-Report-Analysis project
## problem statement
The objective ofthis project is to analyze Amazon sales data to identify key revenue drivers,top-performing categories and regional sales trends.
## Tools & Techmologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
## Key KPIs
- Total revenue
- total Orderds
- Average Order Value
- Top category
- Top state
## Data Analysis Performed
- Data cleaning & Preprocessing
- Category-wise Order Analysis
- Top selling products identification
- Monthly Sales Trend Analysis

## Visualizations
1] Category wise Revenue

2] State-wise Orders

## Key Insights
- Maharashtra is the top-performing state contributing the highest reveneu.
- Certain categories generate high orders but lower revenue.
- Sales show variation across different regions.

## Business Recommendations
- Focus marketing efforts on high performing states.
- Improve pricing strategy for low-revenue categories.
- Promot top-selling products to increase revenue.
